# Joybuy Tone of Voice Guide — Benelux (BE/NL/LU)

## Important: Multi-language Region

The Benelux market spans multiple languages and cultures. Always confirm the target language per campaign:
- **Flanders (Dutch-speaking Belgium)** — shares traits with Netherlands but Belgian Dutch (Flemish) is softer
- **Wallonia + Luxembourg (French-speaking)** — aligns with French conventions but Belgian French is less formal than Parisian French
- **Brussels** — officially bilingual, always produce both NL and FR versions

Never use a single language version for the whole Benelux region without checking the target audience.

## Tone Principles

- **Dutch (BE/NL): Direct & plain-speaking** — same principles as Netherlands Dutch, though Belgian Dutch can be slightly softer in tone
- **French (BE/LU): Warm & elegant** — same principles as French, though Belgian French is slightly less formal than Parisian French

## Currency & Formatting

- All Benelux markets use EUR (€). Decimal separator: comma (€2,00)
- Belgium/Luxembourg: symbol before amount with no space — €2,00
- Netherlands: same — €2,00
- Dutch (BE) quotes: "example" or 'example'
- French (BE/LU) quotes: « example » — guillemets with non-breaking spaces
- Exclamation marks (NL): Very sparingly
- Exclamation marks (FR): Space before ! and ? — e.g. 'Invitez un ami !'

## Things to Avoid

| Avoid | Use Instead |
|---|---|
| One language for all Benelux | Always separate NL and FR versions per region |
| Parisian French tone in Belgium | Belgian French is warmer and less formal — adjust accordingly |
| Netherlands Dutch = Belgian Dutch | Belgian Dutch (Flemish) is softer — avoid overly blunt phrasing |
| Hype language in either language | Both markets value honesty — avoid 'amazing', 'incredible' etc. |
| Assuming Brussels = Dutch only | Brussels is officially bilingual — always produce both NL and FR |
